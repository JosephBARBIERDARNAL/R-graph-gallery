Create Interactive Chord Diagrams with chorddiag

The chorddiag package in R is designed to create interactive chord diagrams using the D3 JavaScript library.
This post showcases the key features of chorddiag and provides a set of graph examples using the package.

{chorddiag}

Quick start

The chorddiag package in R is designed to create interactive chord diagrams using the D3 JavaScript library.

It offers a set of functions that make it easy to create and customize chord diagrams for visualizing relationships between entities.

Installation

To get started with chorddiag, you can install it from GitHub using the devtools package:

# install.packages("devtools")
devtools::install_github("mattflor/chorddiag")

Basic usage

The chorddiag package allows you to create chord diagrams using the chorddiag() function.

Here‚Äôs a basic example:

library(chorddiag)

m <- matrix(
   c(
      11975, 5871, 8916, 2868,
      1951, 10048, 2060, 6171,
      8010, 16145, 8090, 8045,
      1013, 990, 940, 6907
   ),
   byrow = TRUE,
   nrow = 4, ncol = 4
)
groupNames <- c("black", "blonde", "brown", "red")
groupColors <- c("#000000", "#FFDD89", "#957244", "#F26223")
interactive_plot <- chorddiag(m, groupNames = groupNames, groupColors = groupColors)

# save the plot as an HTML file
htmlwidgets::saveWidget(interactive_plot, "../HtmlWidget/chord_diagram-1.html")

Key features

‚Üí Customizing Colors

You can customize the colors of your chord diagram using the groupColors parameter.

Example:

interactive_plot <- chorddiag(m, groupNames = groupNames, groupColors = c("blue", "yellow", "green", "red"))

# save the plot as an HTML file
htmlwidgets::saveWidget(interactive_plot, "../HtmlWidget/chord_diagram-2.html")


‚Üí Adjusting Margins

Use the margin parameter to adjust the margins of your chord diagram.

Example:

interactive_plot <- chorddiag(m, groupNames = groupNames, margin = 100)

# save the plot as an HTML file
htmlwidgets::saveWidget(interactive_plot, "../HtmlWidget/chord_diagram-3.html")


‚Üí Customizing Labels

You can customize the labels of your chord diagram using the groupnamePadding and groupnameFontsize parameters.

Example:

interactive_plot <- chorddiag(m, groupNames = groupNames, groupnamePadding = 20, groupnameFontsize = 12)

# save the plot as an HTML file
htmlwidgets::saveWidget(interactive_plot, "../HtmlWidget/chord_diagram-4.html")


‚Üí Adding Tooltips

chorddiag allows you to add tooltips to your chord diagram using the tooltipGroupConnector parameter.

Example:

interactive_plot <- chorddiag(m, groupNames = groupNames, tooltipGroupConnector = " to ")

# save the plot as an HTML file
htmlwidgets::saveWidget(interactive_plot, "../HtmlWidget/chord_diagram-5.html")
