Introduction to reactable


This post explains how to build an interactive table with the reactable package. It provides several reproducible examples with explanation and R code.

Correlogram section Data to Viz

Default table with reactable

Simply pass a table or dataframe as the first argument to the reactable function.

library("reactable") # load the package
data(iris) # load the iris dataset

reactable(iris)

Column grouping and aggregating

Use the groupBy argument to group rows by a column. The column will be hidden and a toggle will appear above the table to show and hide the groups.

reactable(iris,
          groupBy = "Species", # Group by the Species column
          columns = list(
            Sepal.Length = colDef(aggregate = "max"), # max value
            Sepal.Width = colDef(aggregate = "mean"), # mean value
            Petal.Length = colDef(aggregate = "min"), # min value
            Petal.Width = colDef(aggregate = "sum") # sum value
            )
        )

Custom theme

Use the theme argument to customize the appearance of the table. You can also use the striped or highlight arguments to add stripes or highlight the hovered row.

reactable(
  iris[1:30, ],
  striped = TRUE,
  highlight = TRUE,
  bordered = TRUE,
  theme = reactableTheme(
    borderColor = "black",
    stripedColor = "gold",
    highlightColor = "lightblue",
    cellPadding = "8px 12px"
  )
)

Conclusion

This post explains how to build an interactive table with reactable. This package can offer much more than what is presented here.

I encourage you to check the reactable page to learn more about it.
